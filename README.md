# CCCrochet
A university project on generating simple crochet patterns for coral-esque freeform shapes

> Check out the live version here! -> https://99-knots.github.io/CCCrochet/

> If you crocheted any of the patterns please feel free to fill out this questionnaire: https://forms.gle/Sugdt8ZeussWZRXPA
